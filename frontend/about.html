Problem Statement

Choosing a movie shouldn't be the hardest part of relaxing — yet streaming services only show what they personally own. Postgres & Chill breaks that limitation by aggregating movie data from a large dataset and intelligently recommending films based on unbiased ranking algorithms.

Proposed Solution

The platform imports, cleans, and stores movie and rating data inside PostgreSQL, provides a REST API through Spring Boot, and calculates recommendations using weighted rating formulas and user preferences. New users go through a cold start rating process to personalize results quickly.

Implementation Overview

The system uses a full-stack approach:

  • Java Spring Boot backend with REST endpoints
  • PostgreSQL database for all movie, rating, and user data
  • HTML/CSS/JavaScript frontend with dynamic rendering
  • Docker for easy deployment and consistent environment setup

Technology Stack

  • Backend: Java (Spring Boot), PostgreSQL
  • Frontend: HTML, CSS, JavaScript
  • DevOps: Docker, Docker Compose

Expected Outcomes

  • Full PostgreSQL movie dataset with relationships and indexing
  • API for movies, genres, ratings, and recommendation logic
  • Interactive dashboard with carousels and rating UI
  • Cold-start onboarding for new users
  • Future-ready architecture for neural recommendation models

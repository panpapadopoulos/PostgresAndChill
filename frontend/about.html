What is Postgres & Chill?

Postgres & Chill is a full-stack, fully Dockerized movie recommendation platform powered by PostgreSQL, Spring Boot, and the MovieLens dataset. The goal: deliver personalized, data-driven movie suggestions using collaborative filtering.

Visit: https://github.com/panpapadopoulos/PostgresAndChill

Why This Project?

Most streaming platforms recommend only what they own. But real discovery requires a broader, unbiased movie catalog. This system uses a **real dataset**, **statistical ranking**, and **user behavior** to recommend movies beyond a single platform.

Deployment (Docker)

The entire system runs via Docker Compose:

  • PostgreSQL container (auto-loads MovieLens data)
  • Spring Boot backend container
  • Static frontend served locally

System Architecture

The architecture consists of three layers working together under Docker:

  • Frontend: HTML/CSS/JS with Fetch API
  • Backend: Spring Boot REST services
  • Database: PostgreSQL + MovieLens data

How Recommendations Work

üß† Collaborative Filtering + Weighted Correlation
  • Build rating vector for each user
  • Compare unrated movies to rated ones
  • Similarity is calculated using weighted correlation
  • Movies above user's average = likely liked
  • Movies below user's average = likely disliked
